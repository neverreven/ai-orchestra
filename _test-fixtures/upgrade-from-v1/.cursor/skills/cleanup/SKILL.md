# Cleanup

> Audit recently changed files for unused imports, dead code, and inconsistencies.

## Trigger

- "cleanup"
- "clean up the mess"
- "tidy up"

## When to use

When the codebase has accumulated small messes after a feature implementation.

## Process

1. Read recently changed files.
2. Identify unused imports, dead variables, duplicate logic.
3. Remove or consolidate the issues.
